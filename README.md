# Mini-Url

Mini-Url is a small end to end application for making your own short Url formate with the short url gennerator.

## Project Tools:
## Frontend:
[Vue.js](https://vuejs.org/),
[Vue-Material](https://vuematerial.io/),
[Vuelidate](https://vuelidate.js.org/)
JavaScript,
Sass
Axios requests.
